# Changelog

## [0.0.1] - 2026-04-23

### 새 기능
- 반려동물 프로필 등록 및 관리 (localStorage 기반)
- 의료 기록 및 건강 스케줄 확인
- 규칙 기반 건강 리포트 생성 및 PDF 다운로드
- 예방접종 간격 계산 (강아지/고양이)
- 예약 접수 페이지 및 예약 확인 화면
- 동물병원 소개 페이지 (수의사, 시설, 운영시간, 연락처)
- 서비스 안내 페이지 (진료 항목, 절차, FAQ, CTA)
- 랜딩 페이지 (통계, 이용 후기, 위치 및 연락처 안내)

### 개선
- 전체 UI에 따뜻한 디자인 시스템 일괄 적용
- 위치 안내에 지도 앱 연동 링크 및 클릭 가능한 연락처 추가
