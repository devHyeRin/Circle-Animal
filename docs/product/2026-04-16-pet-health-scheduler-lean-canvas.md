# 반려동물 건강 스케줄러 — Lean Canvas

**날짜:** 2026-04-16  
**단계:** MVP

---

| 칸 | 내용 |
|----|------|
| **문제** | 1. 다음 접종일을 기억하지 못해 놓침 <br> 2. 진료 기록이 여러 곳에 흩어져 조회 불가 <br> 3. 수의사 상담 시 이력 전달이 어려움 |
| **고객 세그먼트** | 강아지·고양이를 키우는 반려동물 보호자 <br> (별도 앱 설치 없이 바로 쓰고 싶은 사람, 수의사 방문 전 기록을 정리해야 하는 사람) |
| **고유 가치 제안** | "로그인 없이 바로, 진료 기록 입력하면 접종 일정 자동 계산 + 수의사 상담용 AI 리포트까지" |
| **솔루션** | 구조화된 진료 기록 입력 폼 (날짜·항목 필수) <br> 규칙 기반 접종 다음 일정 자동 계산 <br> Claude API 기반 수의사 상담용 요약 리포트 생성 |
| **채널** | 첫 100명: 반려동물 네이버 카페(강아지·고양이 커뮤니티) 후기 공유 <br> 이후: 인스타그램 반려동물 계정 바이럴 |
| **수익 모델** | MVP 단계: 없음 (포트폴리오) <br> 이후 검토: 월정액 구독 (다기기 동기화·알림 기능) |
| **비용 구조** | Claude API 호출 비용 (리포트 생성 건당) <br> 서버 호스팅 (Express API 서버) |
| **핵심 지표** | 리포트 생성 횟수 (MVP 목표: 월 50회) <br> 재방문율 (목표: 2주 내 재방문 30%) |
| **경쟁 우위** | 로그인 없는 즉시 사용 경험 <br> 규칙 기반 정확한 접종 계산 + LLM 상담 리포트의 조합 <br> 특정 병원 종속 없이 어디서나 기록 가능 |
